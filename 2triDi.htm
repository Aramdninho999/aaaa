EVALUACION DE SOLUCIONES WEB: AF ANDRÉS INTRIAGO TIENDA DEPORTIVA

FACILIDADES DE USO

La página web de AF Andrés Intríago Tienda Deportiva se destaca por su enfoque centrado en el usuario. A continuación, se detallan las facilidades de uso proporcionadas:

Aspecto Descripción
Navegación La interfaz intuitiva permite a los usuarios encontrar fácilmente los productos deseados.
Menús Los menús están organizados de manera lógica, facilitando la exploración de las diferentes secciones.
Búsqueda La función de búsqueda eficiente mejora la experiencia del usuario al encontrar rápidamente productos específicos.

INTERACTIVIDAD

La interactividad en la página web es una de sus principales características. A continuación, se describen las funcionalidades interactivas implementadas:

Funcionalidad Descripción
Búsquedas Avanzadas Los usuarios pueden realizar búsquedas avanzadas para encontrar productos específicos según criterios detallados.
Filtrado de Resultados Se ha implementado un sistema de filtrado para permitir a los usuarios refinar los resultados según sus preferencias.
Chat en Vivo La integración de un chat en vivo permite la comunicación directa con el equipo de ventas para consultas instantáneas.

CAPTURAS DE PANTALLA

Menú Principal

El menú principal fue diseñado para ser intuitivo y fácil de navegar. Se implementó con HTML y CSS para la estructura y el diseño, y JavaScript para interactividad como desplegables y animaciones.

Interfaz de Usuario

La interfaz de usuario se centra en proporcionar una experiencia atractiva. Los elementos visuales se crearon con CSS, y la interactividad se mejoró con JavaScript para crear transiciones suaves.

Tecnologías Utilizadas

En esta captura se reflejan las tecnologías clave utilizadas en el desarrollo, incluyendo HTML para la estructura, CSS para el estilo y diseño, y JavaScript para la dinámica del sitio.

Diseño Responsivo

Se implementó un diseño responsivo para garantizar una experiencia consistente en dispositivos móviles. CSS media queries fueron esenciales para lograr este objetivo.

Pasos de Instalación

Esta captura muestra los pasos de instalación de la página utilizando Joomla y WordPress, destacando el proceso de descarga, configuración y puesta en marcha.
